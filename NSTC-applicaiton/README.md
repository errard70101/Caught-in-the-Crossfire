## üéØ Success Tips Based on Advisor's Examples

### **Research Proposal**
1. **Be mathematically rigorous** (if applicable)
   - Include detailed model specifications
   - Show derivations step-by-step
   - Reference equations clearly

2. **Separate work by year clearly**
   - Year 1: Core methodology development
   - Year 2: Extensions and applications

3. **Include comprehensive simulation plans**
   - Specify data generating process
   - Multiple sample sizes
   - Clear evaluation metrics

### **Budget**
1. **Personnel**: Provide detailed justification for each position
2. **Equipment**: Justify necessity for research
3. **Travel**: Connect to research dissemination strategy
4. **Materials**: Be specific about usage and need

### **Publications**
1. List target journals explicitly
2. Timeline for submissions
3. Connection to research objectives
4. Past publication record (representative works)
